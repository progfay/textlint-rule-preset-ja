# `@progfay/textlint-rule-preset-ja`

[![npm version](https://img.shields.io/npm/v/@progfay/textlint-rule-preset-ja?style=for-the-badge&message=NPM&color=CB3837&logo=NPM&labelColor=222222&label=npm)](https://www.npmjs.com/package/@progfay/textlint-rule-preset-ja)

This package is textlint-ja preset for @progfay.

## Usage

```json
{
  "rules": {
    "@progfay/textlint-rule-preset-ja": true
  }
}
```

Or you can use this preset directly:

```sh
textlint --preset @progfay/textlint-rule-preset-ja [file]
```
